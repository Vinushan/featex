# Examples

This directory contains example scripts and notebooks demonstrating how to use FeatEx.

## Basic Workflow

Start with `basic_workflow.py` to see:

- Building point-in-time datasets
- Feature aggregation over time windows
- Common use patterns
- Complete end-to-end workflow

Run it:

```bash
python basic_workflow.py
```
